python - <<'PY'
from pathlib import Path
from PIL import Image, ImageDraw

input_dir = Path("data/demo_api_input")
input_dir.mkdir(parents=True, exist_ok=True)

img = Image.new("RGB", (600, 400), "white")

draw.rectangle((40, 40, 560, 360), outline="gray", width=3)

img.save(input_dir / "demo_api_image.png")
print(f"Created: {(input_dir / 'demo_api_image.png').resolve()}")
PY

13. Troubleshooting
------------------------------------------------------------

If port 8000 is busy:
.\dist\clinical-image-anonymizer-api.exe --host 127.0.0.1 --port 8001

Then use:
http://127.0.0.1:8001/docs

If the API executable is missing:
python -m PyInstaller --clean --noconfirm clinical_image_anonymizer_api.spec

If /docs does not open:
Check whether the API window shows an error.
Check Windows Firewall prompt.
Try http://localhost:8000/docs

If an endpoint returns 422:
The request body does not match the schema.
Use Swagger UI to inspect the required fields.

If an endpoint returns 404:
The path does not exist.
List available endpoints with /openapi.json.
